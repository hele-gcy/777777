# JavaScript 基础练习

## 1. 变量定义

```
=== 变量定义 ===
var 重复声明 username: Bob
let score: 85
const PI: 3.14159
```

- **var**：函数作用域，可重复声明，存在变量提升
- **let**：块级作用域，不能重复声明
- **const**：常量，声明时必须赋值，之后不能重新赋值

## 2. 不同类型变量

```
=== 不同类型变量 ===
字符串 str: "Hello, JavaScript!"
整数: 42, 浮点: 3.14, 负数: -10
布尔 isActive: true, isEmpty: false
undefined: undefined
null: null
数组: [10, 20, 30, 40]
对象: 张三, 28岁, 来自北京

typeof 检测:
typeof str: string
typeof integer: number
typeof isActive: boolean
typeof arr: object
```

- **字符串**：`'Hello, JavaScript!'`
- **数值**：整数、浮点数、负数
- **布尔**：true / false
- **undefined**：声明但未赋值
- **null**：空值
- **数组**：`[10, 20, 30, 40]`
- **对象**：`{ name: '张三', age: 28, city: '北京' }`

## 3. 表达式运算

```
=== 表达式运算 ===
x = 15, y = 4
x + y = 19
x - y = 11
x * y = 60
x / y = 3.75
x % y = 3 (取余)
x ** y = 50625 (幂运算)

n += 5 后 n = 15
n *= 2 后 n = 30

比较运算:
3 == "3": true (宽松相等，类型转换)
3 === "3": false (严格相等，不类型转换)
5 > 3 && 5 >= 5: true
5 > 3 || 2 > 8: true
!(3 > 5): true

三元运算: age=20, ticket=成人票
```

- **算术运算符**：+、-、*、/、%、**
- **赋值运算**：+=、-=、*=、/=
- **比较运算**：==、===、>、<、>=、<=、&&、||、!
- **三元运算符**：`condition ? value1 : value2`

## 4. 流程控制（if / for）

```
=== 流程控制 ===

--- if 分支 ---
温度22°C，舒适，穿长袖

--- for 循环 ---
普通 for 循环（1-5）:
  第 1 次循环
  第 2 次循环
  第 3 次循环
  第 4 次循环
  第 5 次循环

for 遍历数组:
  颜色 1: 红
  颜色 2: 绿
  颜色 3: 蓝

for...of 遍历:
  - 红
  - 绿
  - 蓝

for...in 遍历:
  name: 张三
  age: 28
  city: 北京

嵌套循环 - 乘法表:
1×1=1
2×1=2  2×2=4
3×1=3  3×2=6  3×3=9
```

- **if/else if/else**：条件分支
- **for**：普通循环、遍历数组
- **for...of**：遍历元素值
- **for...in**：遍历索引/属性名
- **嵌套循环**：九九乘法表

## 5. 自定义函数

```
=== 自定义函数 ===
add(3, 7) = 10
subtract(10, 4) = 6
max(6, 8) = 8
greet("小明", "早上好") = 早上好，小明！
greet() = 你好，访客！
sumAll(1,2,3,4,5) = 15
数组 [1,2,3,4] 翻倍: [2,4,6,8]
数组 [1,2,3,4] 平方: [1,4,9,16]
factorial(5) = 120 (5! = 5×4×3×2×1)

IIFE 立即执行: 函数在定义时立即运行

=== 练习完成 ===
```

- **普通函数声明**：函数声明会被提升
- **函数表达式**：用 const 定义的函数
- **箭头函数**：`=>` 简写语法
- **默认参数**：参数可以有默认值
- **剩余参数**：`...nums` 收集多个参数
- **回调函数**：将函数作为参数传递
- **递归函数**：函数调用自身（如阶乘）
- **IIFE**：立即执行函数表达式
