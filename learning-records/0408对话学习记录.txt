● 这是个很有价值的问题。让我从几个角度来分析：

  AI 时代的学习理论

  核心原则：
  - 间隔重复 (Spaced Repetition) — 遗忘曲线表明复习间隔递增能最大化长期记忆
  - 主动回忆 (Active Recall) — 从记忆中检索比重读有效得多
  - 刻意练习 (Deliberate Practice) — 专注薄弱环节，而非舒适区
  - 认知负荷理论 (Cognitive Load) — 工作记忆有限，需拆分消化

  AI 时代的优势：
  - 苏格拉底式提问 — AI 可以像导师一样追问，让你从回答中构建理解
  - 即时反馈循环
  - 个性化难度调节
  - 永不疲倦的陪练

  ---
  与 Claude Code Skill 结合的建议

  你的需求是"快速详细掌握某个主题"，我建议一个结构化学习循环的 skill，核心流程：

  1. 诊断 (5分钟)
     → 询问你对该主题的当前认知水平
     → 明确学习目标（能用它做什么？）

  2. 构建框架 (核心)
     → 用 Feynman 技巧：要求你用自己的话解释概念
     → AI 追问细节，发现知识缺口

  3. 间隔复习提醒
     → 将关键问题存入记忆/笔记
     → 设定复习时间线（次日→3天→7天）

  4. 实践验证
     → 给出真实问题让你解决
     → 观察输出，纠正理解偏差

  核心机制

  AI 取代的不是"职业"，是"任务"（Tasks）。

  职业 = 一堆任务的组合

  [任务A] [任务B] [任务C] [任务D] [任务E]

  [X]      [X]     [ ]      [ ]      [ ]
