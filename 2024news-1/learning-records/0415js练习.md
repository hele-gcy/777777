# JavaScript 基础语法练习

## 1. 变量定义（不同类型）

```
=== 变量定义（不同类型）===
字符串 name: Alice
数值 age: 25
常量 PI: 3.14159
布尔 isStudent: true
数组 fruits: [苹果, 香蕉, 橙子]
对象 person: Bob, 男, 爱好: 读书、游泳
typeof name: string
typeof age: number
typeof isStudent: boolean
typeof fruits: object
```

- **var**：函数作用域，可重复声明
- **let**：块级作用域，不可重复声明
- **const**：常量，必须初始化，不可重新赋值

## 2. 表达式运算

```
=== 表达式运算===
加法: 10 + 3 = 13
减法: 10 - 3 = 7
乘法: 10 * 3 = 30
除法: 10 / 3 = 3.333...
取余: 10 % 3 = 1
幂运算: 10 ** 3 = 1000

count++ = 5, 最终 count = 6
++count2 = 6, 最终 count2 = 6

比较运算:
5 == "5": true  (宽松相等，类型转换)
5 === "5": false (严格相等，类型不转换)
3 > 2 && 3 >= 2: true
3 > 5 || 3 < 10: true
!true: false

三元运算符: score=78, 结果=及格
```

- **算术运算**：+、-、*、/、%、**
- **自增/自减**：count++、++count
- **比较运算**：==、===、>、<、>=、<=、&&、||、!
- **三元运算符**：`condition ? value1 : value2`

## 3. 流程控制（if / for / switch）

```
=== 流程控制===

--- if 分支 ---
现在是 14 点，下午好！

--- for 循环 ---
水果 1: 苹果
水果 2: 香蕉
水果 3: 橙子

for...of 遍历数组:
  - 苹果
  - 香蕉
  - 橙子

for...in 遍历对象:
  username: Bob
  gender: 男
  hobbies: 读书、游泳

--- switch 分支 ---
今天是星期三
```

- **if / else if / else**：条件分支
- **for**：遍历数组
- **for...of**：遍历数组元素
- **for...in**：遍历对象属性
- **switch**：多分支选择

## 4. 自定义函数

```
=== 自定义函数===
add(3, 5) = 8
multiply(4, 6) = 24
divide(10, 2) = 5
divide(10, 0) = 除数不能为0
greet("小明") = 你好, 小明！
greet() = 你好, 访客！
sum(1, 2, 3, 4, 5) = 15
数组 1,2,3,4,5 平方后: [1, 4, 9, 16, 25]
factorial(5) = 120
factorial(0) = 1

=== 练习结束 ===
```

- **函数声明**：会被提升
- **函数表达式**：`const multiply = function(a, b) {...}`
- **箭头函数**：`const divide = (a, b) => {...}`
- **默认参数**：`greet(username = '访客', greeting = '你好')`
- **剩余参数**：`sum(...numbers)`
- **回调函数**：将函数作为参数传递
- **递归函数**：函数调用自身（阶乘）
